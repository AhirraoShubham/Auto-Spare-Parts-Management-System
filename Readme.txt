AUTO SPARE PARTS MANAGEMENT SYSTEM

Following are the steps required to run the project :

	JDK 1.6 or higher is required.
	Please copy the "Auto Spares" folder in "C:\b3".
	Change the path (if required) of JDK bin in the file "Run.bat".
	Create the DSN with name "Auto spare".
		DSN can be created from Control Panel - Administrative Tools - Data Sources (ODBC).
	The project can be run either by executing the "Run.bat" file or by running the program
	"Load.java"
